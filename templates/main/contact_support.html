About the Web Application

DRDO DARPAN Portal is a unified digital platform designed to streamline and manage research projects under the DRDO Industry Academia Centres of Excellence (DIA-CoEs). The portal enables seamless collaboration between DRDO laboratories, academic institutions, and industry partners, fostering innovation and accelerating the development of advanced defence technologies.

Key Features:

  • Centralized project database with advanced search and analytics
  • Role-based access for admins, managers, and viewers
  • Secure file uploads and document management
  • Automated reminders for project deadlines and meetings
  • Comprehensive data visualization and reporting tools

Technology Stack:

  • Backend: Python, Flask, Flask-SQLAlchemy, Flask-Login, Flask-Migrate
  • Frontend: HTML5, CSS3, Bootstrap 5, Jinja2 templating
  • Database: SQLite
  • Other Libraries: WTForms, ReportLab (for PDF export), Chart.js (for analytics)
  • Authentication: Role-based access control using Flask-Login
